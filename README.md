# FireFlow Product Intelligence

An independent, interactive portfolio project by **Nathan J. Song**, built to support an application for **Manager, Customer Experience** at Samyang America. It demonstrates product knowledge, consumer and vendor support design, escalation and resolution workflows, transparent measurement, continuous improvement, and the front-end craft to work credibly with technology, ERP, EDI, and data teams.

> FireFlow CX is an independent portfolio concept created from publicly available information. It is not affiliated with, commissioned by, or connected to Samyang America or Samyang Foods. All customers, orders, shipments, complaints, employees, metrics, financial values, lot codes, and outcomes shown are fictional.

## Three connected layers
- **Explore** — discover and understand Samyang's public U.S. portfolio (45 product families).
- **Resolve** — turn a product-specific consumer or vendor question into a verified, routed, governed case.
- **Command** — aggregate cases into a manager's operating picture and into portfolio improvement.

## Stack
React + Vite + TypeScript, custom token-driven CSS. Local synthetic data; structured to add a backend later. See `docs/homepage/MASTER-FIREFLOW-HOMEPAGE-BUILD-PLAN.md`.
